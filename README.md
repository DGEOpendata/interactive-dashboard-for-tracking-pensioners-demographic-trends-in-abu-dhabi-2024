# Interactive Dashboard for Pensioners' Demographic Trends in Abu Dhabi 2024

This repository contains the code to create an interactive dashboard for visualizing the demographic trends of pensioners in Abu Dhabi for the year 2024. The dashboard utilizes the dataset titled "Pensioners Distribution for 2024" and provides user-friendly visualizations to explore gender distribution, quarterly trends, and demographic insights.

## Features
- Bar charts for gender distribution per quarter.
- Line graphs for total pensioners per quarter.
- Export options for visualized data in CSV and JSON formats.
- Built-in data dictionary and methodology summary for clarity.

## Prerequisites
- Python 3.7 or higher
- Required Python libraries:
  - Flask
  - Pandas
  - Plotly
  - openpyxl

## Installation
1. Clone this repository:
   bash
   git clone https://github.com/example-user/pensioners-dashboard.git
   
2. Navigate to the project directory:
   bash
   cd pensioners-dashboard
   
3. Install the required libraries:
   bash
   pip install -r requirements.txt
   

## Usage
1. Place the dataset files in the root directory:
   - `Distribution_of_Pensioners_per_Gender.xlsx`
   - `Distribution_of_Pensioners.xlsx`
2. Run the Flask application:
   bash
   python app.py
   
3. Open your web browser and navigate to `http://127.0.0.1:5000` to access the dashboard.
